📖 O que é Git e GitHub?

🔷 Git

Sistema de controle de versão distribuído criado por Linus Torvalds em 2005. Permite que desenvolvedores acompanhem mudanças no código, revertam versões e trabalhem em equipe de forma eficiente.

  • ✅ Versionamento local
  • ✅ Branching e merging
  • ✅ Rápido e leve

🔶 GitHub

Plataforma de hospedagem de código que utiliza Git como base. Além de armazenar repositórios, oferece ferramentas de colaboração, como issues, pull requests e actions.

  • ✅ Repositórios remotos
  • ✅ Colaboração em equipe
  • ✅ Integração com CI/CD

⚙️ Instalação

🖥️ Git

  1. Acesse git-scm.com
  2. Baixe a versão para seu sistema
  3. Execute o instalador
  4. Deixe as opções padrão
  5. Verifique: git --version

📁 Git Desktop

  1. Acesse desktop.github.com
  2. Baixe e instale
  3. Faça login com sua conta GitHub
  4. Clone ou crie repositórios
  5. Interface amigável para iniciantes

🌐 GitHub Online

  1. Acesse github.com
  2. Crie uma conta gratuita
  3. Verifique seu e-mail
  4. Crie seu primeiro repositório
  5. Explore a interface

⬇️ Do GitHub Online para o PC

1

Crie um repositório no GitHub

Clique em "New", dê um nome e marque "Add a README"

2

Clone o repositório

Copie a URL do repositório (HTTPS ou SSH)

3

Clone via terminal

git clone https://github.com/seu-usuario/repo.git

4

Ou use o Git Desktop

Clique em "Clone a repository" e cole a URL

📤 Do VS Code para o GitHub Online

1

Crie um projeto no VS Code

Abra sua pasta de projeto e comece a codar!

2

Inicialize o Git

git init no terminal do VS Code

3

Adicione e comite

git add . e git commit -m "Primeiro commit"

4

Conecte ao GitHub

git remote add origin https://github.com/seu-usuario/repo.git
git push -u origin main

5

Publique no GitHub

Seu código já está no GitHub. Ative o GitHub Pages nas configurações do repositório para publicar.
